# TokenCourt - Web3 Project Evaluation System

A multi-agent system for evaluating Web3 and blockchain projects through structured roundtable discussions.

## Project Structure
- `backend/`: FastAPI backend service
- `frontend/`: React frontend application

## Features
- 9 AI agents with specialized Web3/blockchain expertise
- Real-time roundtable discussions
- Majority voting system
- Discussion history tracking
- Web interface for viewing discussions and results
